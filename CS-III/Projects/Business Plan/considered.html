Castaway

lilly-seeley

Castaway is an app that allows users to create travel plans easily by planning your travel cost, time, distance, activities, food, and stay on one page. This idea helps plan out all aspects of traveling from one place. I eventually decided against this project since there are similar websites out there, that actually pull prices and deals from their own partnerships and APIs that we don’t have access to. Essentially this project won’t be as appealing as its competitors.

ShelterSafe

onik-hoque

ShelterSafe aims to eliminate the problem of lack of awareness regarding shelters by providing a hub that uses GPS signal to locate the user, and then uses cellular data to convey statistics from each nearby shelter concerning food supply, available space, and distance from each shelter. I like this project since it has a real world use and can save lives during natural disasters.
